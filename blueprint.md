# 프로젝트 개요

"저녁 메뉴 뭐먹지?" 고민을 해결해주는 간단한 웹 페이지입니다. 사용자가 버튼을 클릭하면, 미리 정의된 저녁 메뉴 리스트에서 랜덤으로 하나를 추천해줍니다.

# 스타일 및 디자인

-   전체적으로 깔끔하고 미니멀한 디자인을 추구합니다.
-   중앙 정렬을 사용하여 사용자의 시선을 집중시킵니다.
-   부드러운 색상과 그림자 효과를 사용하여 시각적인 편안함을 줍니다.
-   반응형 디자인으로 모바일에서도 보기 좋습니다.

# 기능

-   **메뉴 추천 버튼:** 클릭 시 저녁 메뉴를 추천합니다.
-   **추천 메뉴 표시:** 추천된 메뉴를 화면에 보여줍니다.

# 현재 요청 계획

1.  `index.html` 파일 생성: 페이지의 기본 구조를 정의합니다.
2.  `style.css` 파일 생성: 페이지의 시각적 스타일을 정의합니다.
3.  `main.js` 파일 생성: 메뉴 추천 기능을 구현합니다.
